What is this?
Sdatch is a casual visualization library based on D3.js, the world's most excellent and popular visualization solution by @mbostock and observablehq.
This is a demo site for sdatch's fundamental visualization with layered chart set. Now sdatch is supplied in alpha state, but it is desired to grow up to one of the major visualization framework with minimalistic features.
It is implemented as an ES module and can be seamlessly integrated to React, VueJS and so on. Users can generate layered figure with a few configuration, with just specifying basic attributes within FigConfig object(s).
Example for smoothed area chart

    let fig = sdatch.createFigure("target-elem-id", 300,300)
    fig.addLayer({
        id: "layer-1",
        type: "area",
        column: ["a","b","c","d","e","f","g"],
        data: [23000, 14500, 15050, 22700, 25503, 23431, 19989],
        axe: {
            x: true,
            y: true,
        },
        animation: false,
        smooth: true,
        color: {
            stroke: "orange",
            fill: "orange"
        },
    }).renderAll()
